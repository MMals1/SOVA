# План изучения ресурсов

## Этап 1 — Основы криптографии
Теория без которой сложно понять остальное.

- [ ] [Cryptography for Developers](https://cryptography-for-devs.github.io) — введение в криптографию для программистов
- [ ] [Cryptography for Programmers (статья)](https://dev.to/shierve/cryptography-for-programmers-1-basics-block-cryptography-1iei) — блочная криптография, основные понятия
- [ ] [CryptoBook by Nakov](https://cryptobook.nakov.com) — полная книга: хеши, симметричное/асимметричное шифрование, ECC

---

## Этап 2 — Эллиптические кривые и ECDSA
Математика за приватным/публичным ключом и подписью транзакций.

- [ ] [Learn Me a Bitcoin — ECDSA](https://learnmeabitcoin.com/technical/cryptography/elliptic-curve/ecdsa/) — подробно про secp256k1 и ECDSA
- [ ] [Building secp256k1 from scratch in Python](https://dev.to/mouiz/building-bitcoins-secp256k1-curve-and-ecdsa-from-scratch-in-python-5ho3) — практика: реализуем кривую руками
- [ ] [Building ECC from scratch](https://medium.com/@abhiveerhome/building-elliptic-curve-cryptography-ecc-from-scratch-7b28e3b27531) — ещё одна практическая реализация

---

## Этап 3 — Построение кошелька
Прикладное применение: от мнемоники до транзакции.

- [ ] [How to Build a Crypto Wallet (Chainlink)](https://chain.link/tutorials/how-to-build-a-crypto-wallet) — туториал по построению кошелька
- [ ] [HD Wallet на GitHub](https://github.com/bethanyuo/HD-wallet) — пример реализации HD-кошелька, изучить код
- [ ] [Ethers.js v6 Docs](https://docs.ethers.org/v6/) — документация библиотеки для работы с Ethereum
- [ ] [Crypto Wallet Development Guide](https://www.walletfinder.ai/blog/crypto-wallet-development) — обзор архитектуры кошельков

Закрепление через решение задач.

- [ ] [Coursera: Cryptography (Dan Boneh)](https://www.coursera.org/learn/crypto) — академический курс, можно проходить параллельно
